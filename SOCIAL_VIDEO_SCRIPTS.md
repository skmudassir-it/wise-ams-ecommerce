# TIKTOK / INSTAGRAM REEL — VIDEO SCRIPT TEMPLATES

## 3 Proven Formats for Product Videos

### Format 1: "The Problem Solver" (15 sec)
```
0-3s:  Show the PROBLEM (frustrated face, messy scene)
3-8s:  Product reveal + quick demo solving problem
8-12s: Satisfied reaction shot
12-15s: Price reveal + "Link in bio!" text overlay
```

### Format 2: "The Unbox" (20 sec)  
```
0-5s:  Package arrives, satisfying unbox
5-10s: First look + close-up of key feature
10-15s: Quick test/use + genuine reaction
15-20s: Price + where to buy
```

### Format 3: "Before & After" (12 sec)
```
0-3s:  BEFORE shot (chaos/pain/problem)
3-4s:  Quick cut transition
4-9s:  AFTER shot (organized/solved/happy) with product in frame
9-12s: Text overlay: product name + price
```

### Caption Formula (Proven CTR):
"Hook (first 3 words matter most) → Pain point → Solution → Price → CTA"

Example: "Stop scrubbing your pans. 🔥 These silicone utensils... (continue)"
